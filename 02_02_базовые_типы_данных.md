## –°–∫–∞–ª—è—Ä–Ω—ã–µ —Ç–∏–ø—ã

–°–∫–∞–ª—è—Ä–Ω—ã–µ —Ç–∏–ø—ã –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –æ–¥–∏–Ω–æ—á–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è. Rust –∏–º–µ–µ—Ç —á–µ—Ç—ã—Ä–µ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Å–∫–∞–ª—è—Ä–Ω—ã—Ö —Ç–∏–ø–∞:

### –¶–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–µ —Ç–∏–ø—ã

–¶–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–µ —Ç–∏–ø—ã –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —á–∏—Å–ª–∞ –±–µ–∑ –¥—Ä–æ–±–Ω–æ–π —á–∞—Å—Ç–∏. Rust –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã—Ö —Ç–∏–ø–æ–≤ —Ä–∞–∑–ª–∏—á–Ω–æ–π —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç–∏ –∏ —Å–æ –∑–Ω–∞–∫–æ–º –∏–ª–∏ –±–µ–∑:

| –î–ª–∏–Ω–∞ | –°–æ –∑–Ω–∞–∫–æ–º | –ë–µ–∑ –∑–Ω–∞–∫–∞ |
|-------|-----------|-----------|
| 8-–±–∏—Ç | `i8` | `u8` |
| 16-–±–∏—Ç | `i16` | `u16` |
| 32-–±–∏—Ç | `i32` | `u32` |
| 64-–±–∏—Ç | `i64` | `u64` |

–ö–∞–∂–¥—ã–π —Ç–∏–ø –∏–º–µ–µ—Ç —Å–≤–æ–π –¥–∏–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω–∏–π:
- –¢–∏–ø—ã —Å–æ –∑–Ω–∞–∫–æ–º –º–æ–≥—É—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç -(2^(n-1)) –¥–æ 2^(n-1)-1, –≥–¥–µ n ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –±–∏—Ç
- –¢–∏–ø—ã –±–µ–∑ –∑–Ω–∞–∫–∞ –º–æ–≥—É—Ç —Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å–ª–∞ –æ—Ç 0 –¥–æ 2^n-1

–ù–∞–ø—Ä–∏–º–µ—Ä:
- `i8` –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç -128 –¥–æ 127
- `u8` –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç 0 –¥–æ 255

–¢–∏–ø—ã `isize` –∏ `usize` –∑–∞–≤–∏—Å—è—Ç –æ—Ç –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã –∫–æ–º–ø—å—é—Ç–µ—Ä–∞: 64 –±–∏—Ç–∞ –Ω–∞ 64-–±–∏—Ç–Ω—ã—Ö —Å–∏—Å—Ç–µ–º–∞—Ö –∏ 32 –±–∏—Ç–∞ –Ω–∞ 32-–±–∏—Ç–Ω—ã—Ö —Å–∏—Å—Ç–µ–º–∞—Ö. –û–Ω–∏ –æ—Å–æ–±–µ–Ω–Ω–æ –ø–æ–ª–µ–∑–Ω—ã –ø—Ä–∏ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏ –∫–æ–ª–ª–µ–∫—Ü–∏–π.

–¶–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–µ –ª–∏—Ç–µ—Ä–∞–ª—ã –º–æ–∂–Ω–æ –∑–∞–ø–∏—Å—ã–≤–∞—Ç—å –≤ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ñ–æ—Ä–º–∞—Ç–∞—Ö:


–ü–æ —É–º–æ–ª—á–∞–Ω–∏—é, –µ—Å–ª–∏ —Ç–∏–ø –Ω–µ —É–∫–∞–∑–∞–Ω —è–≤–Ω–æ –∏ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã–≤–µ–¥–µ–Ω –∏–∑ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞, –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –∏—Å–ø–æ–ª—å–∑—É–µ—Ç `i32` –¥–ª—è —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã—Ö –ª–∏—Ç–µ—Ä–∞–ª–æ–≤.

### –¢–∏–ø—ã —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π

Rust –∏–º–µ–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π: `f32` –∏ `f64`, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç—É IEEE-754:

### –õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø

–õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø `bool` –º–æ–∂–µ—Ç –∏–º–µ—Ç—å —Ç–æ–ª—å–∫–æ –¥–≤–∞ –∑–Ω–∞—á–µ–Ω–∏—è: `true` –∏–ª–∏ `false`. –û–Ω –∑–∞–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω –±–∞–π—Ç –≤ –ø–∞–º—è—Ç–∏.

## –°–æ—Å—Ç–∞–≤–Ω—ã–µ —Ç–∏–ø—ã

### –ö–æ—Ä—Ç–µ–∂–∏ (Tuples)

–ö–æ—Ä—Ç–µ–∂ ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–± –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∏ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∑–Ω–∞—á–µ–Ω–∏–π —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤ –≤ –æ–¥–Ω–æ —Å–æ—Å—Ç–∞–≤–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. –ö–æ—Ä—Ç–µ–∂–∏ –∏–º–µ—é—Ç —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—É—é –¥–ª–∏–Ω—É: –ø–æ—Å–ª–µ –æ–±—ä—è–≤–ª–µ–Ω–∏—è –æ–Ω–∏ –Ω–µ –º–æ–≥—É—Ç —É–≤–µ–ª–∏—á–∏–≤–∞—Ç—å—Å—è –∏–ª–∏ —É–º–µ–Ω—å—à–∞—Ç—å—Å—è.

```rust
    // –î–µ—Å—Ç—Ä—É–∫—Ç—É—Ä–∏–∑–∞—Ü–∏—è –∫–æ—Ä—Ç–µ–∂–∞

    println!("–ó–Ω–∞—á–µ–Ω–∏—è: x = {}, y = {}, z = {}", x, y, z);
    
    // –î–æ—Å—Ç—É–ø –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º –∫–æ—Ä—Ç–µ–∂–∞ —á–µ—Ä–µ–∑ –∏–Ω–¥–µ–∫—Å
    let five_hundred = tup.0;
    let six_point_four = tup.1;
    let one = tup.2;
    
    println!("–ß–µ—Ä–µ–∑ –∏–Ω–¥–µ–∫—Å—ã: {} {} {}", five_hundred, six_point_four, one);
}
```

–ö–æ—Ä—Ç–µ–∂ –±–µ–∑ –∑–Ω–∞—á–µ–Ω–∏–π, `()`, —è–≤–ª—è–µ—Ç—Å—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º —Ç–∏–ø–æ–º, –∫–æ—Ç–æ—Ä—ã–π –∏–º–µ–µ—Ç —Ç–æ–ª—å–∫–æ –æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —Ç–∞–∫–∂–µ –∑–∞–ø–∏—Å—ã–≤–∞–µ–º–æ–µ –∫–∞–∫ `()`. –≠—Ç–æ—Ç —Ç–∏–ø –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –µ–¥–∏–Ω–∏—á–Ω—ã–º —Ç–∏–ø–æ–º (unit type), –∞ –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ ‚Äî –µ–¥–∏–Ω–∏—á–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º (unit value). –í—ã—Ä–∞–∂–µ–Ω–∏—è –Ω–µ—è–≤–Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –µ–¥–∏–Ω–∏—á–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –µ—Å–ª–∏ –æ–Ω–∏ –Ω–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∫–∞–∫–æ–µ-–ª–∏–±–æ –¥—Ä—É–≥–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

### –ú–∞—Å—Å–∏–≤—ã

–ú–∞—Å—Å–∏–≤ ‚Äî —ç—Ç–æ –∫–æ–ª–ª–µ–∫—Ü–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞ —Å —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π –¥–ª–∏–Ω–æ–π. –í Rust –º–∞—Å—Å–∏–≤—ã —Ä–∞–∑–º–µ—â–∞—é—Ç—Å—è –≤ —Å—Ç–µ–∫–µ, –∞ –Ω–µ –≤ –∫—É—á–µ.

```rust
fn main() {
    // –ú–∞—Å—Å–∏–≤ —Å —è–≤–Ω—ã–º —É–∫–∞–∑–∞–Ω–∏–µ–º —Ç–∏–ø–∞ –∏ –¥–ª–∏–Ω—ã
    let a: [i32; 5] = [1, 2, 3, 4, 5];
    
    // –ú–∞—Å—Å–∏–≤, –≥–¥–µ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ

    println!("–ü–µ—Ä–≤—ã–π –º–∞—Å—Å–∏–≤: {:?}", a);
    println!("–í—Ç–æ—Ä–æ–π –º–∞—Å—Å–∏–≤: {:?}", b);
    
    // –î–æ—Å—Ç—É–ø –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º –º–∞—Å—Å–∏–≤–∞

    let second = a[1];
    
    println!("–ü–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç: {}", first);
    println!("–í—Ç–æ—Ä–æ–π —ç–ª–µ–º–µ–Ω—Ç: {}", second);
}
```

–ü–æ–ø—ã—Ç–∫–∞ –¥–æ—Å—Ç—É–ø–∞ –∫ —ç–ª–µ–º–µ–Ω—Ç—É –º–∞—Å—Å–∏–≤–∞ –∑–∞ –µ–≥–æ –ø—Ä–µ–¥–µ–ª–∞–º–∏ –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ –ø–∞–Ω–∏–∫–µ –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è:

```rust

    let a = [1, 2, 3, 4, 5];
    
    // –°–ª–µ–¥—É—é—â–∞—è —Å—Ç—Ä–æ–∫–∞ –≤—ã–∑–æ–≤–µ—Ç –ø–∞–Ω–∏–∫—É –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è
```

–≠—Ç–æ –æ–¥–Ω–∞ –∏–∑ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–µ–π –±–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç–∏ Rust: –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ —Å—Å—ã–ª–∫–∏ –Ω–∞ –º–∞—Å—Å–∏–≤ –≤—Å–µ–≥–¥–∞ –¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã.

## –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Ç–∏–ø–æ–≤

Rust —Ç—Ä–µ–±—É–µ—Ç —è–≤–Ω–æ–≥–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è —Ç–∏–ø–æ–≤ –≤ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ —Å–ª—É—á–∞–µ–≤. –≠—Ç–æ –¥–µ–ª–∞–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é –∫–ª—é—á–µ–≤–æ–≥–æ —Å–ª–æ–≤–∞ `as`.

```rust

    let b = 2.0;
    
    // –Ø–≤–Ω–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ

    // –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Å–∏–º–≤–æ–ª–∞ –≤ —á–∏—Å–ª–æ

    println!("–°–∏–º–≤–æ–ª —Å ASCII-–∫–æ–¥–æ–º 66: {}", e);
```

–í–∞–∂–Ω–æ –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –º–µ–∂–¥—É –Ω–µ–∫–æ—Ç–æ—Ä—ã–º–∏ —Ç–∏–ø–∞–º–∏ –º–æ–∂–µ—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ –ø–æ—Ç–µ—Ä–µ –¥–∞–Ω–Ω—ã—Ö –∏–ª–∏ –Ω–µ–æ–∂–∏–¥–∞–Ω–Ω—ã–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞–º:

```rust

    // –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤ u8 —Å –ø–æ—Ç–µ—Ä–µ–π –¥–∞–Ω–Ω—ã—Ö
    let y = 256 as u8; // –†–µ–∑—É–ª—å—Ç–∞—Ç –±—É–¥–µ—Ç 0 –∏–∑-–∑–∞ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—è
```

## –í—ã–≤–æ–¥ —Ç–∏–ø–æ–≤ –∏ –∞–Ω–Ω–æ—Ç–∞—Ü–∏–∏

Rust –∏–º–µ–µ—Ç –º–æ—â–Ω—É—é —Å–∏—Å—Ç–µ–º—É –≤—ã–≤–æ–¥–∞ —Ç–∏–ø–æ–≤, –∫–æ—Ç–æ—Ä–∞—è —á–∞—Å—Ç–æ –º–æ–∂–µ—Ç –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –Ω–∞ –æ—Å–Ω–æ–≤–µ –µ—ë –∑–Ω–∞—á–µ–Ω–∏—è –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è. –û–¥–Ω–∞–∫–æ –∏–Ω–æ–≥–¥–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —è–≤–Ω–æ —É–∫–∞–∑–∞—Ç—å —Ç–∏–ø:

```rust

    // –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä –≤—ã–≤–æ–¥–∏—Ç —Ç–∏–ø
    let guess = "42".parse().expect("–ù–µ —á–∏—Å–ª–æ!");
```

–ê–Ω–Ω–æ—Ç–∞—Ü–∏–∏ —Ç–∏–ø–æ–≤ –æ—Å–æ–±–µ–Ω–Ω–æ –ø–æ–ª–µ–∑–Ω—ã –≤ —Å–ª–µ–¥—É—é—â–∏—Ö —Å–ª—É—á–∞—è—Ö:
1. –ö–æ–≥–¥–∞ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –Ω–µ –º–æ–∂–µ—Ç –æ–¥–Ω–æ–∑–Ω–∞—á–Ω–æ –≤—ã–≤–µ—Å—Ç–∏ —Ç–∏–ø
2. –î–ª—è —É–ª—É—á—à–µ–Ω–∏—è —á–∏—Ç–∞–µ–º–æ—Å—Ç–∏ –∫–æ–¥–∞
3. –ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å –æ–±–æ–±—â–µ–Ω–Ω—ã–º–∏ —Ç–∏–ø–∞–º–∏

## –ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ç–∏–ø–æ–≤

### –ü—Ä–∏–º–µ—Ä 1: –†–∞—Å—á–µ—Ç –ø–ª–æ—â–∞–¥–∏ –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏–∫–∞

```rust

    // –ò—Å–ø–æ–ª—å–∑—É–µ–º –∫–æ—Ä—Ç–µ–∂ –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏–∫–∞
```

### –ü—Ä–∏–º–µ—Ä 2: –†–∞–±–æ—Ç–∞ —Å —Ç–µ–º–ø–µ—Ä–∞—Ç—É—Ä–æ–π

```rust

    let celsius: f64 = 23.0;
```

## –õ—É—á—à–∏–µ –ø—Ä–∞–∫—Ç–∏–∫–∏

1. **–í—ã–±–∏—Ä–∞–π—Ç–µ –ø–æ–¥—Ö–æ–¥—è—â–∏–π —Ç–∏–ø**: –ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –Ω–∞–∏–±–æ–ª–µ–µ –ø–æ–¥—Ö–æ–¥—è—â–∏–π —Ç–∏–ø –¥–ª—è –≤–∞—à–∏—Ö –¥–∞–Ω–Ω—ã—Ö. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è –Ω–µ–±–æ–ª—å—à–∏—Ö —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `i32` –∏–ª–∏ `u32`, –∞ –Ω–µ `i64` –∏–ª–∏ `u64`, –µ—Å–ª–∏ –Ω–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è –±–æ–ª—å—à–∏–π –¥–∏–∞–ø–∞–∑–æ–Ω.

2. **–ë—É–¥—å—Ç–µ –æ—Å—Ç–æ—Ä–æ–∂–Ω—ã —Å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ–º —Ç–∏–ø–æ–≤**: –Ø–≤–Ω–æ –ø—Ä–µ–æ–±—Ä–∞–∑—É–π—Ç–µ —Ç–∏–ø—ã, –∫–æ–≥–¥–∞ —ç—Ç–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ, –Ω–æ –±—É–¥—å—Ç–µ –æ—Å—Ç–æ—Ä–æ–∂–Ω—ã —Å –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–π –ø–æ—Ç–µ—Ä–µ–π –¥–∞–Ω–Ω—ã—Ö.

3. **–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –∞–Ω–Ω–æ—Ç–∞—Ü–∏–∏ —Ç–∏–ø–æ–≤ –¥–ª—è —è—Å–Ω–æ—Å—Ç–∏**: –î–∞–∂–µ –µ—Å–ª–∏ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –º–æ–∂–µ—Ç –≤—ã–≤–µ—Å—Ç–∏ —Ç–∏–ø, –∏–Ω–æ–≥–¥–∞ –ø–æ–ª–µ–∑–Ω–æ —è–≤–Ω–æ —É–∫–∞–∑–∞—Ç—å –µ–≥–æ –¥–ª—è —É–ª—É—á—à–µ–Ω–∏—è —á–∏—Ç–∞–µ–º–æ—Å—Ç–∏ –∫–æ–¥–∞.

4. **–ò–∑–±–µ–≥–∞–π—Ç–µ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—è**: –ë—É–¥—å—Ç–µ –≤–Ω–∏–º–∞—Ç–µ–ª—å–Ω—ã –ø—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å —á–∏—Å–ª–æ–≤—ã–º–∏ —Ç–∏–ø–∞–º–∏, –æ—Å–æ–±–µ–Ω–Ω–æ –ø—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—é.

5. **–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Å–æ—Å—Ç–∞–≤–Ω—ã–µ —Ç–∏–ø—ã –¥–ª—è –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∏ —Å–≤—è–∑–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**: –ö–æ—Ä—Ç–µ–∂–∏ –∏ –º–∞—Å—Å–∏–≤—ã –ø–æ–ª–µ–∑–Ω—ã –¥–ª—è –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∏ —Å–≤—è–∑–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö, –Ω–æ –¥–ª—è –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö —Ä–∞—Å—Å–º–æ—Ç—Ä–∏—Ç–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä (structs) –∏ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–∏–π (enums).

## –£–ø—Ä–∞–∂–Ω–µ–Ω–∏—è

1. **–†–∞–±–æ—Ç–∞ —Å —Ü–µ–ª—ã–º–∏ —á–∏—Å–ª–∞–º–∏**:
